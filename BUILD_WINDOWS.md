# Сборка Windows приложения

## Требования
- Node.js 18+ (скачать с https://nodejs.org)
- Windows 10/11

## Шаги сборки

### 1. Скачайте проект
Скачайте все файлы проекта в папку на вашем компьютере.

### 2. Установите зависимости
Откройте командную строку (CMD) или PowerShell в папке проекта:
```bash
npm install
```

### 3. Соберите приложение

**Портативная версия (один .exe файл):**
```bash
npm run dist:win
```

**Установщик (.exe с инсталляцией):**
```bash
npm run dist:win
```

### 4. Найдите готовое приложение
После сборки файлы будут в папке `release/`:
- `CarReportGenerator-Portable.exe` - портативная версия (запускается без установки)
- `Генератор Отчётов Авто Setup X.X.X.exe` - установщик

## Использование

### Первый запуск
1. Запустите приложение
2. Перейдите в **Настройки** (шестерёнка внизу слева)
3. Введите **API ключ OpenAI**
4. (Опционально) Укажите URL логотипа для отчётов
5. Нажмите **Сохранить настройки**

### Создание отчёта
1. На главной странице перетащите изображения или нажмите для выбора
2. Загрузите все скриншоты китайского отчёта
3. Нажмите **Сгенерировать отчёт**
4. Дождитесь обработки (1-2 минуты)
5. Готово! Можете просмотреть, скачать или экспортировать

### История
- Все отчёты сохраняются автоматически
- Группировка по автомобилям (VIN)
- Сортировка от новых к старым

## Где хранятся данные?
- Windows: `C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\AppData\Roaming\car-report-generator\`
- Файлы: `reports-data.json`, `settings.json`

## Возможные проблемы

### "Не удаётся проверить издателя"
Это нормально для неподписанных приложений. Нажмите "Подробнее" → "Выполнить в любом случае".

### Антивирус блокирует
Добавьте приложение в исключения антивируса.

### API ключ не работает
1. Убедитесь что ключ начинается с `sk-`
2. Проверьте баланс на https://platform.openai.com
3. Ключ должен иметь доступ к GPT-4 Vision

## Альтернатива: Веб-версия
Если сборка не получается, используйте веб-версию прямо в браузере без установки.
